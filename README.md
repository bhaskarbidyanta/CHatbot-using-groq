# Implementation of a working Chatbot using groq platform

This is a working project of a chatbot made using groq platform using mixtral and llama3
And can generate prompts with different personas like default,expert and creative

Shivan Kumar

### Run these codes 
conda create -p env python=3.10 -y
conda activate env/
pip install -r requirements.txt
pip install langchain_groq
streamlit run app.py
